USER API

1)Parent Registration:
Link: users/parents/registration
Request: POST
Parameters:
	Body:{
		first_name
		last_name
		address
		phone
		email
	}
Responce:
	Confirmation Page

2)Organizer Registration:
Link: users/organizers/registration
Request: POST
Parameters:
	Body:{
		first_name
		last_name
		company_name
		company_details
		address
		phone
		email
	}
Responce:
	Confirmation Page

3)Log In
Link: users/login.jsp
Request: POST
Parameters:
	Body:{
		email:
		password:
	}
Responce:
	index
